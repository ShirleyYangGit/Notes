# 内核态和用户态
CPU - CPL指令级别 -> 0 1 2 3
不同的指令可以跑在不同的级别上。级别比较高的，可以直接访问硬件。比较危险的指令只能比较高的级别才能跑。

Linux操作系统中用了两个级别。
- 内核态 0 - Linux 内核
内存中，有专门的内核空间。内核态可以访问内核空间和用户空间
- 用户态 3 - 用户程序
用户态只能访问用户空间，不能访问内核空间。
